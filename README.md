# Install

Type the follow:
```sh
python3 -m venv venv
```

```sh
source venv/bin/activate
```

```sh
pip install -r requirements.txt
```
And run:

```sh
python manage.py runserver
```
**It is not necessary to do any migration because the project uses SQLite**
